# v1.0 人工验收清单

## 准备
- 已 `bash scripts/build-install.sh`，菜单栏出现 SuperRight 图标
- 已授「完全磁盘访问权限」给 SuperRight.app

## A. 空白处右键（在某文件夹空白处）
- [ ] 新建文件 ▸ 文本类（txt/md/rtf/json/sh）能建出对应空文件
- [ ] 新建文件 ▸ Word/Excel/PPT 模板能建出文件，**且能用对应 App 打开编辑**
- [ ] 新建文件 ▸ 自定义名称… 弹框输入后建出
- [ ] 在此打开 ▸ 终端/Warp/Antigravity 打开当前目录
- [ ] 跳转到 ▸ 各常用目录
- [ ] 复制当前路径（粘贴验证）
- [ ] 显示/隐藏 隐藏文件（Finder 刷新后 . 开头文件显隐切换）

## B. 选中文件右键
- [ ] 复制路径 / 复制名称
- [ ] 移动到 ▸ / 复制到 ▸ 目标
- [ ] 用 App 打开 ▸（选文件点终端 → 进入其所在目录）
- [ ] 工具箱 ▸ 剪切 → 到别处空白右键「粘贴到此」
- [ ] 工具箱 ▸ 压缩为 ZIP / 解压
- [ ] 工具箱 ▸ 转换图片为 ▸ JPG/PNG…（选张图）
- [ ] 工具箱 ▸ 生成二维码（同目录出 .qr.png，手机扫一下是路径）
- [ ] 工具箱 ▸ 文件信息（弹窗显示大小/时间/MD5/SHA256，可复制）
- [ ] 工具箱 ▸ 创建替身 / 创建符号链接
- [ ] 工具箱 ▸ 批量重命名…（多选，填前缀/序号，确认后批量改名）
- [ ] 工具箱 ▸ 永久删除…（弹二次确认，确认后直接消失不进废纸篓）

## C. 文件夹图标（选中一个文件夹）
- [ ] 文件夹图标 ▸ 换颜色 ▸ 各色（图标变色）
- [ ] 文件夹图标 ▸ 设置自定义图标…（选图后图标替换）
- [ ] 文件夹图标 ▸ 拷贝图标 → 另一文件夹 ▸ 粘贴图标
- [ ] 文件夹图标 ▸ 清除自定义图标（恢复默认）

- [ ] 通用：菜单栏图标 / 登录自启 / 使用范围；三个权限&位置按钮能跳转
- [ ] 列表：增删改 新建类型 / 常用目录 / 发送目标 / 打开 App；"选择…/选择 App…" 能选
- [ ] 改完后**右键菜单即时反映**（无需重启）
- [ ] 菜单功能：关掉某分组后，右键对应分组消失

## E. 菜单栏图标
- [ ] 设置… / 打开配置文件 / 在 Finder 显示配置 / 重启 / 退出 均可用
